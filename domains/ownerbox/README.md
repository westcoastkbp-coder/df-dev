# OwnerBox Placeholder Domain

This directory reserves the OwnerBox domain inside DF Core without enabling OwnerBox business logic yet.

- `domain_manifest.json` declares the contour-owned namespace and artifact roots.
- `tasks/` is the placeholder task namespace root.
- `memory/` is the placeholder memory namespace root.
- `artifacts/` is the placeholder artifact root inside the repo tree for future domain wiring and tests.

Current status: structural only.
